### Usage

```
<Card :title="value.title" :content="value.content" :link="value.link"></Card>
```

### Properties

| Option | Type | Default Value | Description |
| ------ | ---- | ------------- | ----------- |
| - title | String | - | Used for the card heading |
| - content | HTML | - | Card content |
| - link | String | - | Card link |
