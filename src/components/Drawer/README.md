### Usage

```
<Drawer label="Open drawer">
  <div class="container">
    <h2>Drawer contentent</h2>
  </div>
</Drawer>
```

### Properties

| Option | Type | Default Value | Description |
| ------ | ---- | ------------- | ----------- |
| label | String | - | Required text for the button |



### Slots

| Option | Default Value | Description |
| ------ | ------------- | ----------- |
| default | - | Will display within the drawer |
