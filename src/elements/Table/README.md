### Usage

```
<Table :fields="fields" :items="items"></Table>
```

### Properties

| Option | Type | Default Value | Description |
| ------ | ---- | ------------- | ----------- |
| items  | Array | - | Table row data passed |
| fields | Array | - | Table header titles  |


### Class modifiers

- **.table--fullwidth** - Prevents the table from stacking on mobile
